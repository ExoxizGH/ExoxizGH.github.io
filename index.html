เกี่ยวกับฉัน

ตลอดช่วงชีวิตของผมตั้งแต่เล็กจนโต ผมมักจะใกล้ชิดกับการเล่นเกมและวงการเกมต่างๆกับพี่ชายตลอด จึงเกิดความสงสัยขึ้นมาและสนใจเลือกเดินเส้นทางของนักพัฒนาเกม ผมมักจะสนุกทุกครั้งถึงแม้จะเริ่มต้นจาก 0 แต่ผมชอบที่จะเรียนรู้ทุกอย่างเกี่ยวกับการสร้างเกมขึ้นมาเกมนึงว่าต้องทำยังไง และอะไรคือสิ่งที่เรียกมันว่าเกมที่สนุกจริงๆ

การศึกษา

Unity C#

ปานกลาง

ผมสนใจและทดลองทำเกมโดยมีประสบการณ์ 3 ปีนับตั้งแต่เริ่ม ในการทำเกมใน Unity C# รูปแบบ 2D และต่อยอดมา 3D ที่ผมรู้สึกสนใจเป็นพิเศษและประการณ์ที่จะหาไม่ได้จากเกม 2D

CLIP STUDIO PAINT

พื้นฐาน

ผมมีวาดรูปของโปรแกรม CLIP STUDIO PAINT เป็นงานอดิเรกรองลงมาจากการสร้างเกมและเล่นเกม ผมสนใจเรื่อง Anatomy และ Art ของตัวละครในเกมมากและวาดมันออกมาในบางครั้ง และมักถูกเลือกให้ช่วยเรื่องของสัดส่วนของตัวละครและการทำ Character sheet

BLENDER

พื้นฐาน

ผมมีทักษะการใช้โปรแกรม BLENDER เบื้องต้นที่หัดเรียนมาเร็วๆนี้ โดยสามารถทดลองปั้นโมเดลหน้าตาตัวละครสไตล์ Anime อย่างเช่น Girls frontline 2 & Zenless Zone Zero ได้

SHADER Programing

พื้นฐาน

ผมเขียน CODE shader ใน Unity เบื่องต้นได้ และผมสามารถปรับค่า shader ให้ Model ตัวละครมีเงาและแสงกระทบตัวละครในสไตล์ Anime ได้
